Install Localhost:

1. Buat database dengan nama db_toko.
2. Extract File pos-kasir.zip yang didownload kedalam directori:
   C:\\wamp\www (jika menggunakan WAMPSERVER).
   C:\\xampp\htdocs (jika menggunakan XAMPP).
   NB: PAstikan anda mengaktifkan Rewrite Module pada webserver Anda.!
3. Buka http://localhost/phpmyadmin. Lalu import db_toko.sql kedalam database yang ada buat.
   NB: file db_toko.sql terdapat pada folder pos-kasir.zip
4. Pada Browser Buka URL http://localhost/pos-kasir/
5. Login dengan
   Username	: admin
   Password	: admin
6. Selesai.
